# UWIN Beverage — SILVER Ag+

Новая версия сайта `uwinbeverage.ru`: продуктовый лендинг SILVER Ag+, отдельный раздел инженерного сервиса, формы обращений и служебные страницы.

## Сервисная точка входа

Для выставочных визиток и QR предусмотрена отдельная ссылка:

```text
https://ivashkov2022-droid.github.io/uwinbeverage-site/#service
```

Ссылка приводит прямо к полноценному разделу инженерного сервиса ГК «Ювин»: «Честный знак», интеграция, пусконаладка, диагностика и поддержка 24/7. Это самостоятельный блок внутри общего сайта, а не переключатель или панель на первом экране.

## Публикация и формы

Для просмотра вёрстки проект не требует сборки: достаточно открыть `index.html` или запустить статический HTTP-сервер из корня проекта.

Для рабочих форм нужен хостинг с PHP и настроенной отправкой почты функцией `mail()`. Обработчики расположены в `api/` и отправляют обращения на `service@uwingroup.ru`.

После успешной отправки каждой формы посетитель получает отдельную страницу «Спасибо» для настройки целей в Яндекс Метрике. Эти страницы:

- закрыты от индексации;
- не кэшируются;
- открываются только в сессии, в которой форма была успешно отправлена;
- при прямом переходе возвращают посетителя к соответствующему разделу сайта.

Политика обработки персональных данных размещена на локальной странице `privacy.html`.

# UWIN Beverage — SILVER Ag+

Независимый прототип новой версии сайта `uwinbeverage.ru`.

Первая итерация содержит три адаптивных экрана:

1. SILVER Ag+ — стабильная вода после розлива;
2. граница контроля завода и действие технологии после розлива;
3. три эффекта: продукт, бренд, производственный контур.

## Сервисная точка входа

Для выставочных визиток и QR предусмотрена отдельная ссылка:

```text
https://uwinbeverage.ru/#service
```

Она автоматически раскрывает инженерный сервис ГК «Ювин»: «Честный знак», интеграцию и поддержку 24/7. Обычный адрес открывает продуктовую историю SILVER Ag+.

## Локальный просмотр

Проект не требует сборки и внешних зависимостей. Откройте `index.html` либо запустите любой статический HTTP-сервер из корня проекта.

## Источники

- действующий сайт `https://uwinbeverage.ru/`;
